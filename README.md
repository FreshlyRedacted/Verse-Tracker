# Verse-Tracker
A super simple play time and kills stat tracker for Star Citizen, coded using Python with the code created entirely by Claude (please dont hate me, I can't code and this should be a feature done by RSI already)

This app does not require any outbound or inbound connections. It does not require you to login to anything. All it does is keep track of specific process times, and automatically tracks only that applications running times (Star Citizen game window)

To add your Kill stats, simply enter the value into the text boxes provided and hit enter on your keyboard 

I've hidden an Easter Egg within the Kill Stats tracker, good luck finding it!
