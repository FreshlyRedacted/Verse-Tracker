# Verse-Tracker
A simple Stat tracker for Star Citizen, coded using Python with the code created entirely by Claude (please dont hate me, I can't code and this should be a feature done by RSI already)

This app does not require any outbound or inbound connections. It does not require you to login to anything. It uses the games Log files stored on your system to track multiple metrics for Server Info

To add your Kill stats and Economy, simply enter the value into the text boxes provided and hit enter on your keyboard 

I've hidden an Easter Egg within the Kill Stats tracker, good luck finding it!

This also features a History tab to allow you to a running history of your game sessions, a summary tab to see all your important stats in one place, an economy section where you can manually enter expenses and sales to track profits, and a server info tab to so you can easily see what server you are on without needing to use r_displayinfo 2
